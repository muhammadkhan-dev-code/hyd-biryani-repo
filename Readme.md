### Description
This project is a web development assignment created for Sir Din Muhammad as part of the Web Engineering course. The assignment includes HTML pages, CSS styling, and image resources for a comprehensive web application.

### Project Structure
- **index.html** - Main entry point of the website
- **css/** - Stylesheet directory containing:
  - about.css
  - contact.css
  - gallery.css
  - global_style.css
  - home.css
  - menu.css
  - recipes.css
- **html/** - HTML pages directory containing:
  - about.html
  - contact.html
  - gallery.html
  - menu.html
  - recipes.html
- **images/** - Image resources directory
